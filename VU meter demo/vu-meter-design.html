Design Features

  • LUFS (blue) - Momentary loudness, updates every ~400ms
  • PEAK (amber) - True peak with hold indicator (white line)
  • Color zones: Green safe, Yellow caution (-12 to -6), Red danger (above -6)
  • Segment lines for professional broadcast look
  • Scale markings: 0, -6, -12, -18, -24, -36 dB
  • Numeric readout below each meter
  • Fits within existing panel width (~90px)
  • Dark theme matches dashboard aesthetic
